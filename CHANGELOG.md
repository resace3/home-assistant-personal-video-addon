# Changelog

## 0.2.0 - 2026-07-10

- Explain missing mounts, indexes, invalid metadata, and unavailable media instead of collapsing every problem into an empty library.
- Lock the runner/viewer contract to Home Assistant's canonical `/share/personal_video_studio` path and runner schema version 1.
- Filter incomplete bundles, harden byte-range parsing, and report the running add-on version accurately.
- Improve Ingress-relative loading, autoplay controls, reduced-motion behavior, responsive settings, and accessibility.
- Publish pinned multi-architecture Debian images and remove the vulnerable legacy Alpine/tempio base.

## 0.1.0 - 2026-07-10

- Initial Home Assistant Personal Video Studio repository and responsive Ingress add-on.
